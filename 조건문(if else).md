

## Swift 기본 문법: 조건문(if else) 

swift의 조건문 중 가장 간단하게 해볼 수 있는 문법은 if...else문 일 것이다. 

기본적인 사용방법은 이렇다.
``` swift
var isIPhoneUser: Bool = true

if isIPhoneUser == true {
    print("아이폰 유저")
} else {
    print("아이폰 유저 아님")
}
//결과 : 아이폰 
```
